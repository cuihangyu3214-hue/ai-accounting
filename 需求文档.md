# AI 智能记账工具 — 需求文档

## 项目概述
一个移动端 AI 智能记账工具，支持语音输入、AI 自动归类、手动录入，以 PWA 形式运行在 iPhone 桌面上。

## 用户场景
- 设备：iPhone 17，Safari 浏览器
- 使用方式：添加到主屏幕，像原生 App 一样打开
- 无服务器，零成本

## 核心需求

### 1. 语音输入
- 点击按钮开始语音录入，说一句话自动转文字
- 基于浏览器 Web Speech API，免费无需服务器

### 2. AI 智能归类
- 语音/文字输入后，调用 AI API 自动识别金额、分类、备注
- 支持配置 API Key（Google Gemini / 国产模型等）
- 用户自己提供 Key，前端直接调用

### 3. 手动填写
- 传统表单模式：金额、分类、日期、备注
- 作为语音/AI 的补充

### 4. PWA
- 支持 Safari「添加到主屏幕」
- 桌面独立图标，全屏运行，无浏览器地址栏
- 离线可用

### 5. 数据存储
- 当前阶段：IndexedDB 本地存储
- 支持导出 CSV（可导入飞书多维表格查看）
- 支持导入/导出 JSON 备份

## 技术方案

### 当前阶段（纯前端）
| 模块 | 方案 |
|------|------|
| 框架 | 纯 HTML/CSS/JS，单页应用 |
| 存储 | IndexedDB |
| 语音 | Web Speech API |
| AI | 前端直调 Gemini 等 API |
| 部署 | PWA，本地文件或托管到 GitHub Pages |

### 架构要点
- **存储层抽象**：数据读写统一走接口，方便后续替换后端

### 后续升级路径
- 加 Cloudflare Worker（免费）做 API 中转
- 实现飞书多维表格自动同步
- AI 调用也走 Worker，更稳定
- 上层代码不用改，只需切换接口地址

## UI 设计

### 风格
- Apple Glass 毛玻璃风格（Glassmorphism）
- 浅色主题，淡紫渐变背景光斑
- 半透明白卡片 + backdrop-filter blur
- 主色 #6C5CE7 紫蓝，渐变 #6C5CE7 → #A78BFA
- 大圆角 18-22px，柔和阴影
- 字体 Inter + PingFang SC

### 页面结构（2页 + 弹窗）

**页面1：记账页（首页）**
- 顶部：大标题"记账" + 月份胶囊按钮 + 三格统计条（本月支出/今日/预算剩余，玻璃卡片）
- 中间：聊天式记录流（iMessage 风格）
  - 用户消息：右侧紫蓝渐变气泡
  - AI 确认：左侧玻璃卡片（分类图标 + 名称 + 金额 + 时间 + 已记录徽标）
  - 按日期分隔，默认最近 7 天，上滑加载更多
  - 长按气泡弹出编辑/删除菜单
- 底部输入栏（毛玻璃）：语音按钮 | 输入框 | + 手动表单按钮
- 底部 Tab：记账 / 统计

**页面2：统计页**
- 顶部：大标题"统计" + 右上角齿轮设置按钮
- 月份总额卡片（渐变玻璃，含月份切换箭头）
- AI 文字总结（玻璃卡片，AI 头像 + 分析文本）
- 分类占比（列表行：图标 + 名称 + 进度条 + 金额百分比）
- 每日趋势（Chart.js 平滑曲线 + 渐变填充）
- 月度对比（上月 vs 本月渐变进度条）
- 账单明细（日/月切换 tab，按日期分组显示每条记录，月视图按天折叠可展开）

**弹窗：手动记账（底部 Sheet）**
- 支出/收入分段控件
- 大号金额输入
- 分类图标网格 4 列（点选高亮 + 紫色边框发光）
- 备注选填 + 日期选择
- 取消/完成按钮

**弹窗：设置（底部 Sheet）**
- AI 模型服务商选择 + API Key 输入
- 月度预算
- 数据导出（CSV/JSON）、导入 JSON
