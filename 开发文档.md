# AI 智能记账工具 — 开发文档

## 技术栈

| 模块 | 技术 | 说明 |
|------|------|------|
| 框架 | 纯 HTML/CSS/JS | 单文件 SPA，无构建工具 |
| 存储 | IndexedDB | 通过抽象存储层访问，方便后续替换 |
| 语音 | Web Speech API | Safari/Chrome 原生支持 |
| AI | Google Gemini API / 可扩展 | 前端直调，用户自配 Key |
| 图表 | Chart.js 4.x | CDN 引入 |
| 部署 | PWA | manifest.json + Service Worker |

## 项目结构

```
AI记账工具/
├── index.html          # 主页面（SPA 入口）

│   ├── icon-192.png

│   └── style.css       # 全局样式（Glass 主题）
├── js/

│   ├── ui.js           # UI 渲染（聊天、统计、表单）

```

## 数据模型

### 记账记录 (Record)

```javascript
{
  id: string,          // 唯一 ID（时间戳 + 随机数）
  type: 'expense' | 'income',
  amount: number,      // 金额，正数
  category: string,    // 分类名称
  note: string,        // 备注
  date: string,        // YYYY-MM-DD
  time: string,        // HH:MM
  createdAt: number,   // 创建时间戳
  source: 'voice' | 'text' | 'manual',  // 输入来源
  rawInput: string     // 原始语音/文字输入
}
```

### 设置 (Settings)

```javascript
{
  aiProvider: 'gemini' | 'qwen' | 'deepseek',
  apiKey: string,
  monthlyBudget: number
}
```

### AI Prompt 设计

**解析输入 Prompt:**
```
你是一个记账助手。请从以下文字中提取记账信息，返回 JSON 格式：
{"type": "expense 或 income", "amount": 数字, "category": "分类名", "note": "简短备注"}

可用支出分类：餐饮、交通、购物、居住、娱乐、饮品、医疗、教育、社交、其他
可用收入分类：工资、奖金、兼职、投资、其他

用户输入："{input}"
```

**月度总结 Prompt:**
```
你是一个理财顾问。请根据以下月度消费数据，用 2-3 句话给出简洁的分析和建议。
月预算：{budget} 元
本月消费数据：{records_summary}
要求：提到总支出、预算使用率、占比最高的分类、与上月的变化趋势、结余预测。
```

## 开发顺序

### Phase 1：基础框架
1. 项目结构搭建，PWA 配置（manifest + SW）
2. IndexedDB 存储层实现
3. CSS 主题（Glass 风格变量 + 通用组件）

### Phase 2：记账页
4. 聊天 UI 渲染（气泡 + AI 卡片 + 日期分隔）
5. 底部输入栏 + 手动表单弹窗
6. 语音识别接入（Web Speech API）
7. AI 解析接入（Gemini API）
8. 记录的增删改操作
9. 顶部统计条实时更新

### Phase 3：统计页
10. 月份切换 + 总额卡片
11. AI 月度总结
12. 分类占比列表
13. 每日趋势图（Chart.js）
14. 月度对比
15. 账单明细列表（日/月切换，按天折叠展开）

### Phase 4：设置 & 数据
16. 设置弹窗（AI 配置、预算）
17. 数据导出 CSV / JSON
18. 数据导入 JSON
19. PWA 离线缓存策略

## 关键实现细节

### 语音识别
```javascript
const recognition = new webkitSpeechRecognition();
recognition.lang = 'zh-CN';
recognition.continuous = false;
recognition.interimResults = true;
```

### IndexedDB Schema
- Database: `ai_accounting`
- Object Store: `records`，keyPath: `id`，索引: `date`, `category`, `type`
- Object Store: `settings`，keyPath: `key`

### PWA manifest.json
```json
{
  "name": "AI 智能记账",
```

### 性能要求
- 首屏加载 < 1s（本地文件）
- 聊天列表虚拟滚动（记录 > 100 条时）
- 图表懒加载（切到统计页时才渲染）
